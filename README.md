# Bottle Cap Detection (Ada Mata Take-Home Test)

Real-time bottle cap color detection (light blue, dark blue, others) with a lightweight ML pipeline designed for edge devices such as Raspberry Pi 5.

## Structure

-   bsort/ → CLI tools
-   notebooks/ → Experiments & analysis
-   configs/settings.yaml → Model & dataset config
-   scripts/ → Utility scripts
-   models/ → Saved weights
-   tests/ → Unit tests
